The Anti-AI Manifesto

Let's unpack this. The subject concedes that AI may be used for debugging — a necessity given the opacity of C compiler errors — but draws a hard line at automated code production, which she dismisses as "slop." This is not a technical position. This is a moral one.

Every project maintained by the subject is implemented in pure C. The build system is make. The editor is whatever ships with the terminal. There are no abstractions. There is no garbage collection. There is only the programmer and the memory, locked in eternal combat.
